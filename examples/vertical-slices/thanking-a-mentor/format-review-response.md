*Reply, posted today, from Data Platform Engineering, Ashgrove Systems:*

Thank you for this, Dana - especially the line about the first two meetings, the detail everyone reading this will skim past. You didn't need to speak after that because you'd already said the harder thing in private, back in March 2016: that we were ready before we believed it ourselves.

The full accounting isn't for a public reply. That part is already running, off the record, the way you ran it with us. Anyone reading this before backing someone unproven can treat it as the second data point.
