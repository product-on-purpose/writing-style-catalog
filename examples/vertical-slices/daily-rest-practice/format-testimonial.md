"My weeks used to run on continuous availability - answer everything, stay ahead of the queue, never actually stop. I'd tried a day off before and it never held; the longest run was six weeks, until a deadline pulled it under. This time I turned it into a rule instead of an intention, something I've started calling rest-day: one day in seven, phone in another room, nothing checked, kept since March. Fourteen weeks in now, and the window away from my phone has grown from four hours to ten. Two problems I'd been circling for days both cleared the Monday after the last one held. If stopping has never worked for you as a choice, try it as a rule instead. That's the only version that's ever stuck for me."

- Daniel Weiss, Operations Analyst, fourteen weeks into rest-day
