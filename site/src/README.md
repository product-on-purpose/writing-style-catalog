# src/

The Astro site source. The documentation site is Astro + Starlight; this folder holds the app code that turns the generated `docs/` tree into the published site.

- `content.config.ts` - mounts the repo-root `docs/` tree as the Starlight `docs` content collection (a custom glob loader), excluding internal and per-folder README files.
- `components/DiffPair.astro` - the side-by-side diff-pair component the generated diff-pair pages import.
- `styles/custom.css` - Starlight theme overrides (accent color and tweaks).

The site is configured in `astro.config.mjs` at the repo root and built with `npm run build` (which runs the generator first, then `astro build`). The content itself is generated by `scripts/generate_site_pages.py`, not authored here. See `REPOSITORY.md` for the full pipeline.
